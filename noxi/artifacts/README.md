# NoXi Reproducibility Artifacts

This directory follows the same two-track pattern as the PinSoRo package.

- `training_pipeline/`: commands and notes for retraining from organizer-provided processed features.
- `inference_only/`: checkpoints, configs, compact metrics, and submitted prediction folders for reproducing the clean W750 NoXi results without retraining.

Large prepared tensors and `val_predictions.csv` frame dumps are intentionally omitted.
