# ACM Reproducibility Package

This repository is organized into two challenge-track families:

- `pinsoro/`: PinSoRo reproducibility artifacts.
  - `pinsoro/CR`: CR submitted models, feature ablations, partner/encoder ablations, and sensitivity artifacts.
  - `pinsoro/CC`: CC submitted models, submitted-checkpoint modality ablations, partner ablations, and a placeholder for the pending CC sensitivity package.
- `noxi/`: NoXi / NoXi-J reproducibility artifacts for the clean W750 submitted/forward models and paper ablations.

Prepared tensors are intentionally not vendored. Each track includes code, manifests/configs, selected checkpoints, compact metrics, and submitted prediction folders where needed for inference-only reproduction.
